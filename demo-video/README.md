# HomeCanvas demo video

A storyboard-driven product demo (~100s, 1080p30) built with [Remotion](https://remotion.dev). Every visual is React/CSS/SVG motion graphics in the app's **Sarvam-inspired light theme** — cool light chrome, white cards, near-black navy ink, indigo accent, soft warm gradient washes, and custom thin-line iconography (`src/components/Icons.tsx`). Includes a real **CSS-3D dollhouse** (no WebGL) that the camera orbits, walks into, and "path-traces" tile by tile. No screenshots, no private data.

The rendered video lives at [`../docs/homecanvas-demo.mp4`](../docs/homecanvas-demo.mp4).

## Storyboard (10 scenes)

| # | Scene | Visual |
|---|---|---|
| 1 | `title` | Wordmark + tagline |
| 2 | `problem` | The "flat PDF" plan sheet |
| 3 | `trace` | Walls tracing over the plan + live isometric 3D rising |
| 4 | `canvas3d` | CSS-3D dollhouse orbit → top-down → walk-in → tour, synced to mode chips |
| 5 | `edit` | Materials, style packs, stair rotate, pillar-delete warning |
| 6 | `ai` | Assistant chat: type → proposal → Apply |
| 7 | `photo` | Photo Mode: the dollhouse resolves tile-by-tile like a real tiled path tracer |
| 8 | `audience` | Homeowners & buyers · Architects · Interior designers · Developers |
| 9 | `privacy` | Local-first: laptop, crossed-out cloud, three claims |
| 10 | `outro` | Email CTA |

Scene lengths derive from the narration audio (`src/durations.json`), so re-recording narration re-times the video automatically.

## Narration

Generated by `scripts/generate-narration.mjs`:

- **Preferred — ElevenLabs** (expressive male Indian-English voice "Aashish", `eleven_multilingual_v2`): set `ELEVENLABS_API_KEY` in the environment. The key is **never stored in this repo**. Override the voice with `DEMO_VOICE_ID`.
- **Fallback — macOS `say`** (no key needed; robotic but free).

Either way the output is `public/narration/<id>.wav` + `src/durations.json`. Caption text lives in `src/narration.json`; the script has a pronunciation-override map (e.g. the email address is spoken, not spelled).

## Rebuild it

```bash
cd demo-video
npm install
ELEVENLABS_API_KEY=... node scripts/generate-narration.mjs   # or omit the key for macOS say
npx remotion studio src/index.ts                             # live preview
npm run render                                               # out/homecanvas-demo.mp4
```
